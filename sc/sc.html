Crafting Software for the Stars

As a Software Engineer at Space Concordia, I integrate state-of-the-art software solutions with aerospace innovation. My work ensures that our spacecraft and rockets operate with precision in the harsh environment of space.

Spacecraft Engineering

Control Data Handling Systems

I engineered a real-time data acquisition and control system using C++ and Python. This robust system processes telemetry from onboard sensors with exceptional reliability and fault tolerance. By integrating Git for version control and Jenkins for continuous integration, I ensured top-quality software delivery.

Tech Stack: C++, Python, Git, Jenkins

Payload Integration Software

I developed advanced modules to interface with diverse payload instruments. This involved implementing high-frequency data parsing and logging systems, as well as integrating RESTful APIs for seamless communication with ground stations.

Tech Stack: Python, Node.js, RESTful APIs, SQL

ADCS Simulation & Algorithm Development

To achieve precise spacecraft orientation, I designed and implemented sophisticated algorithms for Attitude Determination and Control Systems (ADCS). Leveraging sensor fusion techniques and Kalman filters, I developed simulation tools using MATLAB and Python to validate our approach.

Tech Stack: MATLAB, Python, Embedded C

Rocketery Engineering

Flight Performance Analysis Tools

I created an end-to-end simulation and data analysis suite in Python for evaluating rocket flight trajectories. This tool processes telemetry data, performs in-depth statistical analysis, and generates performance reports. It even utilizes machine learning techniques to optimize flight parameters for future missions.

Tech Stack: Python, Pandas, SciPy, scikit-learn

Structural Health Monitoring

I developed software solutions to monitor rocket structural integrity during test flights. By integrating sensor data with Finite Element Analysis (FEA) simulations, I was able to identify stress points and potential failures, streamlining our prototyping and testing cycles.

Tech Stack: Python, MATLAB, FEA Software, SQL

SC-Frayer CubeSat

About SC-Frayer

The SC-Frayer CubeSat is Space Concordia's flagship nanosatellite project, conceived to push the boundaries of space research and innovation. This ambitious project is not an isolated effort—it integrates the robust control systems, precise payload management, and cutting-edge ADCS algorithms I developed in the Spacecraft Engineering phase.

Moreover, the CubeSat’s mission is enhanced by advanced rocketery performance analysis and structural health monitoring techniques, as highlighted in our Rocketery Engineering work. Together, these elements form a seamless and sophisticated software ecosystem that underpins SC-Frayer’s operations.

Explore more about the SC-Frayer CubeSat and Space Concordia’s innovative initiatives on the Space Concordia website.
